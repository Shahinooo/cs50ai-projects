# Degrees — Search Algorithms

## Overview

This project finds the shortest connection between two actors using graph search.

Actors and movies are represented as nodes in a graph, and the algorithm searches for the shortest path between them using Breadth-First Search (BFS).

This project demonstrates how search algorithms can be applied to real-world relationship networks.
