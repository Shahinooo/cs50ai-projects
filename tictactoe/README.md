# Tic-Tac-Toe — Minimax Algorithm

## Overview

This project implements an AI that plays Tic-Tac-Toe optimally using the Minimax algorithm.

The AI evaluates all possible future game states and selects the move that maximizes its chances of winning while minimizing the opponent's advantage.
