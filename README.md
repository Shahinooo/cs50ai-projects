# CS50AI Projects

This repository contains my projects from **CS50's Introduction to Artificial Intelligence with Python** by Harvard University.

The course covers core artificial intelligence concepts including search algorithms, knowledge representation, probability, optimization, machine learning, and neural networks. These projects were implemented using Python and demonstrate practical applications of AI techniques.

This repository reflects my foundational training in artificial intelligence and problem solving using algorithmic thinking.

---

## Topics Covered

- Search Algorithms
- Adversarial Search (Minimax)
- Logic and Knowledge Representation
- Probability and Bayesian Inference
- Optimization
- Machine Learning
- Neural Networks
- Natural Language Processing

---

## Project List

### Search

**Degrees**
- Finds the shortest connection between actors using graph search.

**Tic-Tac-Toe**
- Implements the Minimax algorithm to play optimal Tic-Tac-Toe.

---

### Knowledge

**Knights**
- Uses propositional logic to solve logical puzzles.

**Minesweeper**
- Builds an AI agent that plays Minesweeper using logical inference.

---

### Uncertainty

**Heredity**
- Calculates probabilities of genetic inheritance using Bayesian reasoning.

**PageRank**
- Implements Google's PageRank algorithm using sampling and iteration.

---

### Optimization

**Crossword**
- Solves crossword puzzles using constraint satisfaction algorithms.

---

### Learning

**Shopping**
- Predicts online shopping behavior using machine learning classification.

**Nim**
- Implements reinforcement learning using Q-learning.

---

### Neural Networks

**Traffic**
- Uses a convolutional neural network (CNN) to classify traffic signs.

---

### Language

**Parser**
- Uses natural language processing to parse sentences using context-free grammar.
